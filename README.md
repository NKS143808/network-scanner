# 🌐 Network Scanner

A Python tool that discovers active devices on a local network using ARP requests.

## Features

- Discover devices on a LAN
- Display IP addresses
- Display MAC addresses
- Simple command-line interface

## Technologies

- Python
- Scapy

## Installation

```bash
py -m pip install -r requirements.txt
```

## Run

```bash
py main.py
```

## Example

```
Enter Network Range:
192.168.1.0/24

Active Devices

IP Address : 192.168.1.1
MAC Address: xx:xx:xx:xx:xx:xx
```
